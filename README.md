# comparevi-history

Short-name GitHub Action facade for advanced Compare VI History workflows.

Planned public entrypoint:

```yaml

```

This repository is intended to provide a short external action surface while delegating core implementation to:

- `LabVIEW-Community-CI-CD/compare-vi-cli-action`
